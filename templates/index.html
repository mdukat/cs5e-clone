CS5e

Character Sheet, 5th edition

CS5e is a simple, free, online character sheet manager for use with the Dungeons & DragonsTM 5th edition tabletop roleplaying game.

Save paper and stay organized by storing your character sheets online.

A valid sheet code is 3-5 characters, comprised of uppercase letters and numbers.

CS5e uses JavaScript. Please enable scripts and refresh the page.

It is recommended (but not required) that you access CS5e with the latest version of Chrome or Firefox.

announcement Update: Development hiatus

It's been a while since I posted anything here. The truth is, my vision for version 2 of CS5e is far too ambitious for me to tackle on my own at this time. This version is and has been "good enough" for my own purposes, and I've thus been unable to gather the motivation to push forward with development. My personal life has also been a bit more hectic as of late. So, for now, I've decided to put this project on hiatus. The service will stay online, of course (I'm using it, myself) but little to no development will take place until further notice. I'll pick it up again, with gusto, some time in the future.

That said, I do have an idea bouncing around in my head for a clean-ish way to implement image support for character portraits and organization symbols into the current version, and it may or may not show up in the near future. That was, after all, a planned feature of version 1.

announcement Version 2 in developmentexpand_moreexpand_less

When I first started this project, some choices were made that allowed rapid development at the cost of code quality. While this got the project up and running faster, it meant that eventually those issues would have to be addressed. While I took care to write the code in very generic and modular ways whenever possible, a few of these shortcuts have proven extremely difficult to replace. They have even held up implementation of what was originally supposed to be a core feature: character portrait uploads.

As I have been working to improve CS5e over the past month or so, it has become increasingly clear to me that much of the code simply needs to be rewritten. I always intended a full rewrite at some point, so this isn't completely out of left field. Maintaining data compatibility with this version will be a primary objective of the rewrite, as will reimplementing all current functionality. So, when the new version is ready, you'll be able to bring all your characters with you.

I have no plans to shut down this version of CS5e, so feel free to continue using it while the new version is under development. If you are interested in contributing to the development process as a tester, please use the contact form below to get in touch. At this time, I am not seeking additional developers.

help_outline Frequently Asked Questionsexpand_moreexpand_less

  • Is this really free? Will it always be free?
    • In a word: YES.
    • This project was born out of my love for D&D and my desire to stay more organized across the multiple groups and campaigns I play. This is a tool I use myself, and I gladly provide it as a service to all, free of charge.
  • Can you add _____ feature?
    • CS5e is currently undergoing a rewrite. Going forward, the only updates this version will receive are bugfixes.
  • What if I forget the password for a character sheet?
    • Because CS5e stores passwords as cryptographic hashes, recovering a lost password is impossible.
    • However, if you took the prudent step of setting a password recovery e-mail address for the character sheet in question, you can have the password reset.
      • The alternate_email button in the tools panel build can be used to set your recovery email address.
    • Use the contact form below with the 'Forgot My Password' subject. Be sure to include your character sheet URL and use the same e-mail address as your sheet's recovery address.
